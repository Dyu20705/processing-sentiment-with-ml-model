# Sentiment Analysis from Scratch

Tài liệu hướng dẫn chi tiết cho bộ dữ liệu Stanford Sentiment Treebank trên Kaggle nằm ở [docs/readme.md](docs/readme.md).

Repo này được thiết kế để bạn tự xây pipeline phân tích cảm xúc từ đầu, theo thứ tự: tải dữ liệu, tiền xử lý văn bản, trích xuất đặc trưng, rồi tự hiện thực các mô hình Naive Bayes, Decision Tree, SVM và Neural Network.

Nếu bạn đang dùng script tải dữ liệu hiện có, hãy lưu ý rằng file [download_dataset.py](download_dataset.py) đang trỏ tới bộ dữ liệu IMDB; phần hướng dẫn trong docs tập trung vào competition Stanford Sentiment Treebank.
